Using VSC Packages

  1. For Mac, hold down the Option key. For PC hold down the Ctrl key
  2. Click at the start (or anywhere within) each line that you want to wrap. This will create a cursor at each location you click.
  3. Press Cmd + Shift + P to open the command palette.
  4. In the command palette, type Emmet: Wrap with Abbreviation and select it. (Or select it as the latest command)
  5. Type the tag abbreviation you want to wrap around the selected lines (e.g., div).
  6. Press Enter.
  7. Each line with a cursor will be wrapped with the specified tag.

Installing Live Server in Visual Studio Code

One of the most key elements of success in learning this material is constantly checking your work. After adding a few lines of code, you will want to see what your deployed page looks like. While it is possible to intermittantly refresh your browser, it makes more sense to use a VSC extension that will refresh it every time you save your code.

  1. Open the Extensions View:
    1. Click on the Extensions icon on the sidebar (it looks like a square with four squares inside it).
    2. Alternatively, you can press Ctrl + Shift + X on Windows/Linux or Cmd + Shift + X on Mac to open the Extensions view.
  2. Search for Live Server:
    1. In the search bar at the top of the Extensions view, type Live Server.
    2. The search should return the “Live Server” extension by Ritwick Dey.
  3. Install Live Server:
    1. Click on the Install button next to the Live Server extension.
  4. Wait for Installation:
    1. Wait a few moments while the extension is downloaded and installed.

Using Live Server

  1. Open Your HTML File:
    1. Open the HTML file you want to view live in your default browser OR open the entire folder of the file you want to open in Visual Studio Code.
  2. Start Live Server:
    1. Right-click on the HTML file in the Explorer sidebar and select Open with Live Server.
    2. Alternatively, you can press Alt + L, Alt + O (on Windows/Linux) or Cmd + L, Cmd + O (on Mac) to start Live Server.
  3. View Your HTML File in the Browser:
    1. Your default web browser should automatically open and display your HTML file.
    2. Any changes you make to the HTML file in Visual Studio Code will be automatically reflected in the browser without needing to refresh the page.

Stopping Live Server

  1. To stop the Live Right clidk to find the Stop Live server command, find the stop button in the Visual Studio Code status bar at the bottom, or close the browser window.
